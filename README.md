## What I Do

**📋 Business Analysis**  
End-to-end requirements elicitation, process mapping, stakeholder management, and solution design across enterprise IT programs. BRD, RTM, SIPOC, swimlane diagrams, UAT planning, and change impact assessment — delivered at CBAP standard.

**🗂️ Data Governance**  
Enterprise data governance frameworks — RACI-based data ownership models, metadata management, data dictionaries, data classification policies, and access governance. Experienced with Microsoft Entra ID, Purview, and IAM governance at scale.

**📊 Data Analytics & Power BI**  
Designing and delivering executive analytics dashboards that replace manual reporting and enable real-time decision-making. Experienced in Power BI data modeling (star schema), DAX, row-level security, and Azure Data Factory pipelines.

**⚙️ Lean Six Sigma — DMAIC**  
Full DMAIC lifecycle delivery — Define through Control — for enterprise process improvement programs. Fishbone analysis, 5-Why, Pareto, process capability, control charts, and sustained improvement governance.

**🏗️ IT Project & Program Management**  
Delivering complex, multi-vendor, multi-stakeholder IT programs using Agile, Waterfall, and hybrid methodologies. Risk management, stakeholder engagement, vendor governance, and executive reporting — PMP standard.
